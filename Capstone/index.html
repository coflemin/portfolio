Problem Description

The swine industry is a vital part of the agricultural sector, where biosecurity is essential for animal health and industry success. The Rapid Access Biosecurity (RAB) appâ„¢, developed by the Machado Lab at North Carolina State University, addresses these challenges by providing critical biosecurity information. The capstone project aimed to enhance the RABapp by creating a comprehensive spatial representation of intra-barn layouts, transforming sketches and tabular data into a digital format for better visualization and understanding.

Analysis Procedures

This project involved a multifaceted technical approach, combining data management with advanced visualization tools. Key steps included organizing diverse data sources, employing Geopackage for CSIA spatial data, and creating a Web App using ArcGIS Online. The project focused on developing a barn-subdividing script to automate the creation of detailed spatial representations within the RABapp.

Results

The project successfully integrated enhanced spatial data into the RABapp, providing a user-friendly web application for visualizing intra-barn layouts.
